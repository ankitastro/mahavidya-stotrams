================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Artwork rejected
SOLUTION: Check it's exactly square (1:1 ratio), RGB mode, no text/URLs

PROBLEM: File upload fails
SOLUTION: Ensure files are WAV or FLAC, not MP3. Check file isn't corrupted.

PROBLEM: Don't have artwork yet
SOLUTION: Use Canva (free) or hire on Fiverr ($5-20). Search "album cover design"

PROBLEM: Unsure about track order
SOLUTION: Use the 1-10 order from the metadata JSON (Kali first, Kamala last)

PROBLEM: Title too long
SOLUTION: DistroKid allows up to 200 characters, but some stores truncate at 100
